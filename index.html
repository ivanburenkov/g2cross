This software can calculate second-order correlation function G(2)(t) using photon detection times from two single-photon detectors stored in a PicoQuant .PTU file (HydraHarp/MultiHarp/...). Work on other data formats (e.g. produced by Swabian instruments time taggers .ttbin) is in progress.
This web tool is using parts of C++ code written by S.V. Polyakov.
    If you see an empty graph after calculation is complete please click "Calculate" button again.

Second-order correlation function G(2)(t)

Please use sidebar to choose file in .ptu (Hydraharp/Multiharp) format.

Known issues: For large files (>>1 MB), you may need to repeat the calculation to get data displayed in the graph. This code was tested with files up to 100 MB. Note that using larger files may result in excessive memory use and crush your browser.
